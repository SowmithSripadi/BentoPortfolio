# Bento Portfolio

This is a portfolio website inspired by bento design. Bento design finds inspiration in Japanese Bento boxes, which use rounded rectangles to form dynamic layouts.

---

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Build Tool:** Vite
